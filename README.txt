# B1 Trading Dashboard

A clean, mobile-friendly dashboard for B1 Trading Wallet activation.

## Features
- Modern responsive design
- Display of user wallet information
- Secure redirect to B1 Wallet conversion page

## Usage
1. Open `index.html` in your browser.
2. Click the "CONVERT" button to proceed to your wallet.
